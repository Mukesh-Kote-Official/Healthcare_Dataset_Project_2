# 🏥 Healthcare Patient Analytics Dashboard

## 📌 Project Overview
The **Healthcare Patient Analytics Dashboard** is an interactive Power BI project designed to analyze hospital patient data. This dashboard provides valuable insights into patient demographics, medical conditions, hospital performance, billing, admissions, and treatment outcomes.

The goal of this project is to help healthcare professionals and hospital management make data-driven decisions through interactive visualizations.

---

## 🎯 Objectives

- Analyze patient admissions and discharge trends
- Monitor hospital performance
- Track patient demographics
- Analyze medical conditions and treatments
- Compare billing amounts across hospitals
- Improve healthcare decision-making using data

---

## 📊 Dashboard Features

### KPI Cards
- Total Patients
- Total Billing Amount
- Average Billing Amount
- Average Age
- Average Length of Stay

### Visualizations
- Patient Count by Medical Condition
- Billing Amount by Hospital
- Gender Distribution
- Admission Type Analysis
- Insurance Provider Analysis
- Blood Type Distribution
- Test Results Analysis
- Age Group Analysis
- Monthly Admission Trend

### Filters (Slicers)
- Hospital
- Gender
- Medical Condition
- Admission Type
- Insurance Provider
- Blood Type
